Alice is capable of creating, searching, and modifying active directory groups.

Supported Services:

  • Creation of the following groups:
    • Account Groups (ACG)
    • Application Access Groups (AGG)
    • Application Deploymenet Groups (APP)
    • Distribution Groups (DG)
    • Data Management Groups (DMDL) 
    • Resource Access Groups (RADL)
    • Service Management Groups (SMDL)
